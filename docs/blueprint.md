# **App Name**: PortPilot CAU

## Core Features:

- Local Authentication: Secure user authentication system with Admin, Customs, and User roles. Admin can create new users and assign passwords.
- Dashboard Overview: Display of operational news (text input by admin) and real-time occupancy indicators for TTP1, TTP2, and auxiliary zones. Allows manual updates to occupancy rates.
- Dynamic Forms: Users can fill out and submit dynamic forms, attaching files (PDF, JPG) such as DUA and transport documents.
- Document Management: An inbox for the operational/customs department to view user requests and download attached documents. Includes access control based on user roles.

## Style Guidelines:

- Primary color: Dark shade of blue (#30475E) to convey trustworthiness and stability, reminiscent of industrial machinery.
- Background color: Very light gray (#F0F0F0), nearly white, for high contrast with content in the light color scheme.
- Accent color: Yellow-orange (#D68A30) for call to actions and highlighting important information.
- Body and headline font: 'Inter', a sans-serif font, for a modern, neutral and objective look suitable for both headlines and body text.
- Use clear, utilitarian icons for navigation and actions, focusing on simplicity and recognizability. Icons to reflect logistics/port operations.
- Dashboard-style layout with clear sections for news, occupancy indicators, and quick access to the CAU module. High-contrast elements for easy readability.
- Subtle transitions and animations for loading states and updates to provide visual feedback without being distracting.
